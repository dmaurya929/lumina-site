# Form Demo Site

This is a demo site for AEM Forms, containing several HTML forms and static pages.

## Structure

- `index.html`: Home page
- `checkout.html`: Checkout form
- `contact.html`: Contact us form
- `feedback.html`: Feedback form
- `inquiry.html`: Inquiry form
- `assets/`: Images and other assets
- `css/`: Stylesheets
- `js/`: JavaScript files

## How to use

This site is hosted on GitHub Pages.
